# Flipkart Clone (Front-End Practice Project)

A static front-end clone of the Flipkart homepage built purely for educational purposes and UI/UX design practice. 

## 🚀 Purpose
This repository is a personal project created to practice semantic HTML structures, advanced CSS alignments, font integration, and custom layout setups. It is entirely static and does not collect any user credentials or host live transactions.

## 🛠️ Tech Stack Used
* **HTML5**: Structured markup for the navigation, product carousels, categories, and banners.
* **CSS3**: Built using modern layouts, exact custom spacing metrics, sticky elements, and cross-device scroll alignments.
* **Font Awesome**: Implemented for clean, scalable navigation and UI vector icons.
* **Google Fonts**: Uses the "Inter" typography setup for premium interface presentation.

## 📦 Features Included
* **Exact UI Matching**: Replicates the precise styling proportions of the e-commerce design template.
* **Sticky Navigation Engine**: The header bar locks elegantly to the top while traversing lower index columns.
* **Fluid Category Slider**: Responsive item wrappers that smoothly adapt horizontally on narrow viewports without breaking icon formats.
* **Product Card Assemblies**: Configured with proper image boundaries and font weight distributions.
