# BookMyShow Clone (Front-End Design Practice)

A static front-end clone of the BookMyShow homepage built for user interface design and layout practice.

## 🚀 Purpose
This project was built to practice structuring rich content cards, media grids, layout filters, and hero carousels. It focus entirely on static presentation layers using semantic markup and vanilla styles.

## 🛠️ Tech Stack Used
* **HTML5**: Semi-semantic components for header tracking, content sliders, sidebars, and localized booking categories.
* **CSS3**: Layout alignments utilizing Flexbox systems, hover scale effects, box-shadow accents, and multi-row layout structures.
* **Font Awesome**: Applied across search boxes, icon filters, and navigation controls.

## 📦 Features Replicated
* **Structured Movie Display Grids**: Proper card ratios with uniform cover scaling, text overflow styling, and tag elements.
* **Filter Sidebars**: Simulated layout components for language, genre, and content format choices.
* **Live Slider Areas**: Hero image dimensions structured to scale naturally based on desktop viewing layouts.
