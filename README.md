# My Web UI Clones Portfolio 🚀

Welcome to my frontend development playground! This repository contains pixel-perfect clones of popular web applications that I built to master HTML5, advanced CSS3 properties, and responsive layout designs.

## 📁 Repository Directory

### 1. Flipkart Clone
* **Status**: Completed ✅
* **Folder Location**: `/Flipkart-Clone`
* **Core Focus**: Replicating modern e-commerce navigation, sticky header structures, and horizontally scrollable category strips.

### 2. BookMyShow Clone
* **Status**: Completed ✅
* **Folder Location**: `/BookMyShow-Clone`
* **Core Focus**: Managing structured movie card layouts, filter pill sidebars, carousels, and grid spacing for event listings.

### 3. YouTube Clone
* **Status**: Completed ✅
* **Folder Location**: `/YouTube-Clone`
* **Core Focus**: Implementation of a collapsible side navigation drawer, fluid video card grids, and top tag filter bars.

## 🛠️ Global Tech Stack Covered
* **Semantic Markups**: Standard HTML5 structural framing.
* **Layout Engine**: Extensive usage of Flexbox alignment patterns and CSS Grid architectures.
* **Typographic Delivery**: System fonts, custom `@import` web fonts, and proportional text scaling.
* **Asset Integration**: High-definition vectorized layout icons via Font Awesome.
