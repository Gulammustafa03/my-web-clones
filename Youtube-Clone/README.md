# YouTube Clone (Front-End Layout Practice)

A static front-end clone of the YouTube homepage dashboard focused entirely on custom layouts, complex flex structures, and responsive grid configurations.

## 🚀 Purpose
This project is built to master collapsible layout mechanics, multi-column configurations, horizontal navigation pill strips, and highly fluid multi-row video card grids. It is a completely static UI presentation layer.

## 🛠️ Tech Stack Used
* **HTML5**: Semantically structured layout components utilizing headers, side navigation bars, main canvas sections, and clean card containers.
* **CSS3**: Implementation of dynamic CSS Grid columns, explicit Flexbox spacing distributions, hover transitions, and sticky placement engines.
* **Font Awesome**: Leveraged for structural sidebar items, navbar actions (create, notifications, apps), and search field button iconography.

## 📦 Features Replicated
* **Collapsible Side Navigation Sidebar**: Main left drawer alignment formatted to compress or scale cleanly alongside the viewport.
* **Tag Filter Pill Ribbon**: A clean, horizontally aligned filter tag bar stretching above video feeds with hidden overflow properties.
* **Fluid Video Feed Grid**: Configured using modern fractional grids to seamlessly display clean thumbnail grids across varying viewport sizes, maintaining exact aspect ratios without image warping.
